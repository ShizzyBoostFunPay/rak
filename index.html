-- Lesorub v0.1.7
-- Настройка: Папка: settings → config.txt
                    Раздел [main]:
password — пароль, с которым будут регистрироваться на сервере (буквы/цифры)
wood_limit — максимальное количество древесины (цифры)
                    Раздел [settings]:
hidingEmptyLines — скрытие пустых строчек в интерфейсе программы (true/false)
print_cef — вывод цефов в консоль
banip_exit — закрытие RakSamp Lite при блокировке IP
                    Раздел [telegram]:
notifications — включение уведомлений в Telegram (true/false | если отключено, остальные настройки Telegram игнорируются)
token — токен Telegram-бота (буквы и цифры)
chat_id — ID аккаунта Telegram (цифры)
                    Раздел [coordMaster]:
time — временной интервал работы координатного мастера (цифры, в мс)
step — шаг координатного мастера (цифры)
ms_x/ms_y/ms_z — скорость перемещения по XYZ при работе курдмастера (цифры)

Если в окне отображаются непонятные символы, открой папку settings, файл RakSAMP Lite.ini и поменяй значение параметра win7_fix на противоположное (с 1 на 0 или наоборот). Если проблема останется, попробуй найти в интернете, как изменить кодировку в твоей версии Windows.

Почему RakSamp Lesorub - правильный выбор

Полный цикл работ

Автоматическая рубка леса. Отслеживание лимита ДВК в реальном времени. Автосохранение аккаунта при достижении лимита.   Создание нового аккаунта для непрерывной работы.

Telegram уведомления

Логируется каждое действие, заработок, сохранение и смена аккаунта.

Автосохранение состояния

В момент срабатывания лимита система мгновенно фиксирует текущий профиль, Server:Login:Password.

Ротация аккаунтов

После успешного сохранения автоматически создаётся новый аккаунт, готовый к работе без простоев и ручного вмешательства.

🚀 Запуск за 3 минуты

1

Установи RakSamp Lite

Скачай последнюю версию эмулятора клиента SA-MP. Настрой прокси для многопоточного фарма.

3

Настрой параметры

Открой: settings выбери config.txt - настрой пароль, телеграмм уведомления и т.д.

4

Запуск

Открой: RakLaunch Lite - выбери сервер и сгенерируй ник. Либо запусти все сервера одним кликом: start all servers.bat

Важные файлы: config.txt main.lua !rak_protect libs
